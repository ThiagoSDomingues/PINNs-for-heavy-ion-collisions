SA_PINN_ACTO notebook

✅ What the cell does
• It evaluates the trained model on a uniform 200 × 200 grid.
• It extracts the physical fields q, α, and n (computed with the correct EOS).
• It saves everything to a timestamped .npz file that contains exactly the arrays required for the comparison.
