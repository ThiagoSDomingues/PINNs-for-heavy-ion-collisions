------------------------------------
Loss function instability metric (I)
------------------------------------

A useful instability metric should satisfy a few properties:
вЂў Be small when the loss decreases smoothly.
вЂў Increase when the optimization oscillates.
вЂў Be independent of the absolute scale of the loss.
вЂў Be easy to compare across different PDEs and PINN architectures.

from pathlib import Path
import pandas as pd

root = Path("results")

records = []

for pde in root.iterdir():

    if not pde.is_dir():
        continue

    for run in pde.iterdir():

        history = pd.read_csv(run/"history.csv")

        loss = history["loss"].values

        metric = instability_total_variation(loss)

        records.append({
            "PDE": pde.name,
            "Run": run.name,
            "Instability": metric,
            "Final Loss": loss[-1],
            "L2": history["l2"].iloc[-1],
            "Training Time": history["time"].iloc[-1],
        })

df = pd.DataFrame(records)

-------------------------------
Scatter plots for data analysis
-------------------------------
Once we've built the DataFrame, we need to generate the following plots: to compare L2 error, instability metric, final loss, and training time in a 3D scatter plot:

import seaborn as sns
import matplotlib.pyplot as plt

sns.set_theme(style="ticks", context="paper")

fig = plt.figure(figsize=(8, 7))
ax = fig.add_subplot(111, projection="3d")

sc = ax.scatter(
    df["L2"],
    df["Instability"],
    df["Final Loss"],
    c=df["Training Time"],
    cmap="viridis",
    s=80,
)

ax.set_xlabel(r"$L_2$ Error")
ax.set_ylabel(r"Instability $I$")
ax.set_zlabel(r"Final Loss $L$")

cb = plt.colorbar(sc)
cb.set_label(r"$t_E$ (s)")

plt.tight_layout()
plt.show()

This gives us a single summary visualization of the relationship among accuracy, optimization stability, convergence quality, and computational cost. The same DataFrame can also be used to produce the time vs. instability and time vs. Lв‚‚ error plots, as well as the projection-style figures.
